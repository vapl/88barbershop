# 88 Barbershop

Premium multilingual (lv/en/ru) website for 88 Barbershop Riga built with Next.js, TypeScript, Tailwind CSS, Sanity CMS, and Framer Motion.

## Summary
A brand?forward, CMS?driven site with location?specific services, pricing, and contacts. The content model is structured for easy updates by the team, with responsive layouts and smooth motion across key pages.

## Highlights
- Multilingual content (LV/EN/RU) with Sanity CMS
- Dynamic services and pricing by location
- Custom animations with Framer Motion
- Fully responsive UI

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Sanity CMS
- Framer Motion
